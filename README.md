# Pandas Mastery

An interactive, four-level (ELI5 → expert) tutorial on the three pandas skills that power market-data work: **time series**, **groupby**, and **merging** (including `merge_asof`). Animations, quizzes, and typed exercises with an export-for-grading button. Fully self-contained — no external assets.

Live: https://stranger9977.github.io/pandas-mastery/
