1THE RECIPE — end to end, one verb per line

# ── 0 IMPORTS (pandas trio now; sklearn later, when Block 5 needs it) ──
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

# ── 1 LOOK ──
prices = pd.read_csv("data/prices.csv", parse_dates=["ts"])
print(prices.head())
print(prices.info())

# ── 2 ORDER (windows want within-lane order) ──
prices = prices.sort_values(["ticker", "ts"])

# ── 3 CLEAN DUPS (count what you drop) ──
n0 = len(prices)
prices = prices.drop_duplicates()
print("dup rows dropped:", n0 - len(prices))

# ── 4 RETURNS — three real columns ──
prices["logp"] = np.log(prices["price"])
prices["day"]  = prices["ts"].dt.normalize()
prices["ret"]  = prices.groupby(["ticker", "day"])["logp"].diff()

# ── 5 SPIKE CYCLE ──
print(prices["ret"].abs().nlargest(8))     # look at the top
# np.exp(top value) → the multiplier: 0.69→x2, 2.30→x10 = fat finger
bad = prices["ret"].abs() > 0.5            # threshold IN THE GAP you saw
print("spikes nulled:", bad.sum())
prices.loc[bad, "price"] = np.nan          # .loc to WRITE
# !! RE-RUN the three lines from Block 4 — ret is stale until you do
prices["logp"] = np.log(prices["price"])
prices["ret"]  = prices.groupby(["ticker", "day"])["logp"].diff()
print("max |ret| now:", prices["ret"].abs().max())   # confirm the clean

# ── 6 PLOT SLOT 1 — prove the clean ──
wide = prices.pivot_table(index="ts", columns="ticker", values="price")
wide.plot(subplots=True, figsize=(10, 8))
plt.savefig("plots/prices_clean.png")      # script mode: SAVE it

# ── 7 RECTANGLE — one row per (ticker, day) ──
daily = prices.groupby(["ticker", "day"]).agg(vol=("ret", "std"))
daily = daily.reset_index()
daily["vol"] = daily["vol"] * np.sqrt(390 * 252)     # annualize

# ── 8 HEADLINE ──
print(daily.groupby("ticker")["vol"].mean().sort_values())

# ── 9 INTERVAL — rerun history 2000x, keep the middle 95 ──
v = daily.loc[daily["ticker"] == "QRT", "vol"]
boots = [v.sample(len(v), replace=True).mean() for _ in range(2000)]
lo, hi = np.percentile(boots, [2.5, 97.5])
print(f"QRT ann. vol {v.mean():.3f}, 95% CI [{lo:.3f}, {hi:.3f}]")

Mnemonic: on is the timeline I walk, by is the lane I stay in.
Sort trap: windows want ["ticker","ts"]; merge_asof wants the ON key globally sorted — re-sort by time right before the asof, both sides.

3Model block (only if asked)

from sklearn.pipeline import Pipeline                # the wrapper
from sklearn.preprocessing import StandardScaler     # scaling = preprocessing
from sklearn.linear_model import Ridge               # the model = a linear model
from sklearn.dummy import DummyRegressor             # the baseline = a dummy
from sklearn.model_selection import TimeSeriesSplit, cross_val_score

daily = daily.sort_values(["ticker", "day"])
daily["vol_lag1"] = daily.groupby("ticker")["vol"].shift(1)   # yesterday, per lane
daily["y"]        = daily.groupby("ticker")["vol"].shift(-1)  # tomorrow = target
m = daily.dropna(subset=["vol_lag1", "rating", "y"])
X, y = m[["vol_lag1", "rating"]], m["y"]

model = Pipeline([("scale", StandardScaler()), ("ridge", Ridge())])
base  = DummyRegressor(strategy="mean")
cv    = TimeSeriesSplit(n_splits=5)
print("model MAE:", -cross_val_score(model, X, y, cv=cv,
                     scoring="neg_mean_absolute_error").mean())
print("base  MAE:", -cross_val_score(base,  X, y, cv=cv,
                     scoring="neg_mean_absolute_error").mean())
# persistence baseline: MAE of |vol_lag1 - y| — the one to beat
print("persist MAE:", (m["vol_lag1"] - m["y"]).abs().mean())
Honest sentence: "The model {beats / roughly matches} persistence — with this little data I'd trust the baseline until the gap is clearly outside the noise."

5Defense one-liners (say these when asked "why?")

Why…The sentence
log price?"Log returns are additive across time and symmetric up/down — diff() of log price gives me returns in one vectorized line, and at minute scale they're ≈ percent moves anyway."
sort first?"diff() and rolling windows trust row order — sorting ["ticker","ts"] makes each lane chronological so 'previous row' means 'previous minute of the same name'."
groupby before diff?"So the first minute of each ticker/day diffs against nothing instead of another ticker's close — the overnight gap and cross-ticker seams become honest NaNs."
that threshold?"I looked at the tail: real returns top out near X, the bad prints sit at Y — I cut in the gap between them, not at a magic number."
null, not drop?"Nulling keeps the row's timestamp in place so the grid stays intact; the NaN propagates into the two returns that touched it, which is exactly the poison I want removed."
bootstrap?"Rerun history 2000 times, keep the middle 95. It's distribution-free — no normality assumption on vol, which is skewed."
block bootstrap? (if pushed)"Vol clusters day-to-day, so iid resampling understates the width — I'd resample contiguous blocks of days to preserve the autocorrelation."
tolerance=30D?"Ratings arrive roughly every 4 days; 30 days is generous but past that a rating is stale — I'd rather have an honest NaN than a zombie rating."
dedupe quotes/ratings?"Duplicate timestamps make 'most recent' ambiguous — keep=\"last\" makes merge_asof's silent last-wins choice explicit."
no separate holdout?"TimeSeriesSplit IS the holdout here — every fold tests on data after its training window. I'd only add a final untouched slice if I were tuning/selecting, since the winner's CV score runs optimistic."
Ridge?"Two noisy correlated features, tiny data — a little shrinkage beats OLS variance. Scale first so the penalty treats both features fairly."
CI vs PI?"CI is uncertainty about the average; a prediction interval for tomorrow's value is much wider — it adds the day-to-day noise back in."

6Your gremlin watchlist

  • Named agg shape: vol=("ret","std") — equals, paren, quote-comma-quote. No colons, no braces inside .agg(). Colons live in dicts {} only.
  • Methods return new: df = df.sort_values(...), df = df.reset_index() — always reassign.
  • Write with .loc: read df[mask], write df.loc[mask, "col"] = ... (chained df[mask]["col"]= writes to a copy that evaporates).
  • Recompute downstream: cells hold values, not formulas — after nulling prices, re-run logp and ret.
  • transform takes a string: .transform("mean"), not .transform.mean(). agg collapses; transform broadcasts back to every row.
  • Close the tuple before the comma (your finger habit on the first agg tuple).
  • Python spells it True/False, capitalized.
  • head() before you code — never guess column names.
  • Paths: data lives in data/ next to the script; print os.getcwd() if lost.

7Script mode + plots

  • Nothing shows unless you print() it. Print early, print often — each run is your feedback loop.
  • Plots: plt.savefig("plots/name.png"), never rely on plt.show().
  • Structure main.py with the spine as comments first: # 1 INGEST · 2 EDA · 3 CLEAN · 4 FEATURES · 5 MODEL · 6 INTERVAL · 7 REPORT — then fill.
  • The code freezes at the end — the printed REPORT block is the artifact. End with a tidy print of: headline ± CI, traps found (with counts), baseline verdict, "with more time…".

8When stuck (the Day-of moves)

  • Clarify first: "Before I start — what's the one number you want at the end, and can I check assumptions with you as I go?"
  • Error: read the last line of the traceback, then isolate with .head(10) on the exact object.
  • Blank: say know / want / gap out loud: "I have X, I want Y, the gap is Z" — the gap is always one recipe step.
  • Frozen: "Give me ten seconds to collect the plan." Then read your spine comments.
  • Ask early: "I'd normally reach for ___ here — is that the direction you want?" A hint at minute 20 beats silence at minute 50.
  • Halfway check: at ~45 min, cut scope out loud: "I'll lock the headline + CI, then merges, and describe the model if time runs short."
Priority stack if time collapses: clean data → daily vol table → headline + CI → merges → model. A finished smaller thing beats an unfinished bigger thing.
